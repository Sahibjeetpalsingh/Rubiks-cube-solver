Rubik's Cube Solver Website (2D + Animation)
Creators: Sahibjeet Pal Singh • Bhuvesh Chauhan
Inspired by: Kociemba Two-Phase Algorithm

How to run
1) Open a terminal in this folder
2) Compile + run:

Linux/Mac:
  ./run.sh

Windows:
  run.bat

Then open:
  http://localhost:8080

Input formats
- Scramble moves:  R U R' U'
- 9×12 testcase net (like scramble04.txt), keep spaces.
